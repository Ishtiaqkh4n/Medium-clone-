# Medium Blog Web App clone

A **Medium-inspired blog application** where users can create posts, leave comments, and like content. The platform supports secure authentication through OAuth services, enabling seamless sign-up and login. It is built using **Express.js**, **Appwrite**, **MongoDB**, **JWT**, and a **React-based frontend**, ensuring a scalable and modern full-stack architecture.

## Tech Stack

- **Frontend:** React  
- **Backend:** Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT + OAuth  
- **Backend Services:** Appwrite

##  Features  
- User authentication (Sign up / Sign in with OAuth)  
- Create, edit, and delete blog posts  
- Like and comment on posts  
- Secure API with JWT authentication  
- Scalable full-stack architecture
